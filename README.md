#J8

J8 is the new generation of [Jabir Project](http://jabirproject.org)'s CPU's. This one is a 8 bit CPU, and with a larger range of operations. 

## Operation codes

* `00` : Bitwise NOT
* `01` : Bitwise AND
* `10` : Bitwise OR
* `11` : ADD
